# Root3301

Portfolio website for root3301 - Security researcher and penetration tester.

## Features

- Modern React + TypeScript stack
- Terminal-themed UI with hacker aesthetic
- Dark/Light mode support
- Interactive terminal emulator
- Smooth scroll navigation
- Easter eggs (Konami code support)
- Responsive design with Tailwind CSS
- Custom animations and effects

## Development

Install dependencies:
```bash
npm install
```

Start development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Technologies

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
- **Supabase** - Backend (configured)

## Keyboard Shortcuts

- `Ctrl + 1-5` - Navigate to sections
- `↑↑↓↓←→←→BA` - Konami code

## Terminal Commands

Type these in the interactive terminal:
- `help` - Show available commands
- `about` - Learn more about root3301
- `skills` - View technical skills
- `projects` - Browse projects
- `contact` - Get contact information
- `clear` - Clear terminal
- `ls` - List directories
- `whoami` - Display user information
- `date` - Show current date and time
